🌾 Crop & Fertilizer Recommendation System

An intelligent web application designed to assist in agricultural decision-making by providing data-driven recommendations for crops and fertilizers based on soil and environmental conditions. This project leverages Machine Learning to optimize agricultural yield by analyzing key parameters like NPK levels, pH, and rainfall.

✨ Key Features

•Crop Recommendation: Predicts the most suitable crop to grow based on specific environmental and soil parameters.
•Fertilizer Suggestion: Recommends the optimal fertilizer for specific soil conditions to enhance productivity.
•User-Friendly Interface: A clean, responsive web UI for easy data entry and instant visualization of results.
•ML Integration: Seamlessly processes user inputs through pre-trained Machine Learning models via a robust Flask backend.

🛠️ Tech Stack

•Frontend: HTML5, CSS3, Bootstrap
•Backend: Python, Flask
•Data Science: Scikit-learn, Pandas, NumPy
•Analysis: Jupyter Notebook for Exploratory Data Analysis (EDA) and model development.

📂 Project Structure

•app.py: The core Flask application handling routing and model integration.
•static/: Contains visual assets and styling (CSS/Images).
•templates/: HTML templates for the web interface.
•Research_Notebook.ipynb: Comprehensive documentation of the data analysis and model training process.
•Data_core.csv: The primary dataset used for model reference and analysis.

💡 Functionality Overview

The system allows users to input various environmental and soil parameters to receive instant, data-driven recommendations. By bridging the gap between complex Machine Learning models and a user-friendly interface, it makes advanced agricultural insights accessible for practical use.
