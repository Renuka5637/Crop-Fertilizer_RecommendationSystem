# 🌾 Crop & Fertilizer Recommendation System

---

### 🚀 Core Functionalities

| Feature | Technical Implementation |
| :--- | :--- |
| **Crop Prediction** | Random Forest model analyzing NPK, pH, and environmental factors. |
| **Fertilizer Suggestion** | Predictive engine based on soil nutrients and selected crop type. |
| **Web Integration** | Flask-based REST API for seamless frontend-backend communication. |
| **Data Analysis** | Comprehensive EDA and model training documented in Jupyter. |

---

### 🛠️ Technical Stack

*   **Backend:** Python 3.x, Flask
*   **Frontend:** HTML5, CSS3
*   **Machine Learning:** Scikit-learn, NumPy, Pandas, Pickle
*   **Environment:** Jupyter Notebook

---

### 📂 Repository Structure

*   📁 **`static/`**: Visual assets and custom CSS.
*   📁 **`templates/`**: HTML user interface templates.
*   📄 **`app.py`**: Main application logic and model serving.
*   📄 **`crop_model.pkl`**: Serialized crop recommendation model.
*   📄 **`fertilizer_model.pkl`**: Serialized fertilizer suggestion model.
*   📄 **`Research_Notebook.ipynb`**: ML development and training lifecycle.
*   📄 **`Data_core.csv`**: Project dataset.

---

### 🧪 Input Parameters

*   🌡️ **Environment**: Temperature, Humidity, Rainfall.
*   🌱 **Soil Nutrients**: Nitrogen (N), Phosphorous (P), Potassium (K).
*   🌍 **Soil Characteristics**: Type (Black, Clayey, Loamy, Red, Sandy) and Moisture.
